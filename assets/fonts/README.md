# Fonts

This directory contains the font files used in the application.

Required fonts:
- Poppins-Regular.ttf
- Poppins-Medium.ttf
- Poppins-SemiBold.ttf
- Poppins-Bold.ttf
- Roboto-Regular.ttf
- Roboto-Medium.ttf
- Roboto-Bold.ttf

Please download these font files from Google Fonts and place them in this directory.
